# Project Title: Sales Analysis

# Description:
This project presents a comprehensive Sales Analysis dashboard developed in Power BI to evaluate business performance across multiple dimensions such as sales, profit, customers, delivery efficiency, product categories, and regional performance. The dashboard provides key performance indicators (KPIs), year-over-year comparisons, category-level insights, regional breakdowns, and sales forecasting to support strategic decision-making. It is designed to help stakeholders monitor growth trends, identify high-performing segments, and optimize operational efficiency.

# Project Tools & Components:
Power BI Desktop: Primary data visualization tool used to design and build interactive reports and dashboards. 
Power Query: Used for data cleaning, transformation, and reshaping to prepare raw data for analysis. 
DAX (Data Analysis Expressions): Applied for creating calculated measures, enabling dynamic visuals, and implementing analytical logic.  
File Formats: .pbix used for dashboard development and .png used for sharing dashboard previews.

# Features:
Key KPIs:
Total Sales
Total Profit
Total Customers
Average Delivery Days
Time-Based Analysis:
Monthly Sales Trend
Monthly Sales YoY %
Monthly Profit YoY %
Category & Product Analysis:
Category-wise Sales
Top 5 Subcategories by Sales
Regional & Segment Insights:
Sales by Region
Sales by Ship Mode
Sales by Segment
Interactive Slicers:
Payment Mode
Region
These slicers dynamically update all visuals for focused analysis.

# Business Problem:
The organization lacked a centralized and interactive system to monitor sales performance across different regions, categories, and customer segments. Decision-makers faced challenges in:
Tracking sales and profit growth trends
Comparing year-over-year performance
Identifying top-performing categories and subcategories
Understanding regional and shipping performance
Forecasting future sales for strategic planning
This dashboard solves these problems by providing a unified, dynamic, and data-driven reporting solution that enables better strategic decisions and performance optimization.
